## 🚀 Getting Started

***Requirements***

Ensure you have the following dependencies installed on your system:

* **JavaScript**: `version x.y.z`

### ⚙️ Installation

1. Clone the liwa-store repository:

```sh
git clone https://github.com/liwa-dev/liwa-store.git
```

2. Change to the project directory:

```sh
cd liwa-store
```

3. Install the dependencies:

```sh
npm install
```

### 🤖 Running liwa-store

Use the following command to run liwa-store:

```sh
node app.js
```

### 🧪 Tests

To execute tests, run:

```sh
npm test
```
